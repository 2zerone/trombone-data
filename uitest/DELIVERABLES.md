1. **업무 코드 생성** - TEST001 업무코드 자동 생성
2. **저장소 생성** - TEST-REPO 저장소 자동 생성
3. **사용자 관리** - user001 사용자 및 역할 자동 설정
4. **품질 관리** - SonarQube, JUnit 자동 등록
5. **파이프라인** - STG/PRD 환경 파이프라인 자동 생성
